# PandaJump
Authors:  
  MahSonn  
  Direccc  

Panda Jump - Inspired from the Google Dino Game      

Panda Jump is a simple game where you control a panda to jump and roll to avoid obstacles.  

Try to survive as long as possible while increasing your score.
